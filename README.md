## HOW TO COMPILE
Fract is written in Go and entry point file is 'src/main.go'.
Command are usually like: go build src/main.go

## EXAMPLE CODE
```
var variable.1 int32 := 4
var variable.1 int32 := 4
variable.1 + variable.2 # Printed 8
```
