# print function

## Description
Print values to CLI.

## Define
```
protected func print(value, fin="\n")
```

## Parameters
+ ``value`` <br>
Value to print.
+ ``fin="\n"`` <br>
Value to print after "value" parameter.

## Examples
```
print("Hello World")

# OUTPUT
Hello World
```
```
print("Hello", fin=" ")
print("World")

# OUTPUT
Hello World
```
