# Make Module

Make module used for interpret Fract codes.
<br>

> File extension does not need to be specified, but can be specified if desired.

Examples:
```
> fract make main.fract
[Your Code Output]
```
```
> fract make main
[Your Code Output]
```

Make module is can be directly called with file name. <br>
Examples:
```
> fract main.fract
[Your Code Output]
```
```
> fract main
[Your Code Output]
```
