# ``Ceil`` function

## Description
Returns the smallest (closest to negative infinity).

## Define
```
protected func Ceil(x)
```

## Parameters
+ ``x`` <br>
Numeric.

## Examples
```
open std.math

print(math.Ceil(-23.11)) #  -23
print(math.Ceil(300.16)) #  301
print(math.Ceil(300.72)) #  301
```
