# ``Degrees`` function

## Description
Returns radians to degrees.

## Define
```
protected func Degrees(radians)
```

## Parameters
+ ``radians`` <br>
Radians to parse.

## Examples
```
open std.math

print(math.Degrees(3))  # 171.8873
print(math.Degrees(-3)) # -171.8873
print(math.Degrees(0))  # 0.0
```
