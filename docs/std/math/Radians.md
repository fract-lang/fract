# ``Radians`` function

## Description
Returns degrees to radians.

## Define
```
protected func Radians(degrees)
```

## Parameters
+ ``degrees`` <br>
Degrees to parse.

## Examples
```
open std.math

print(math.Radians(3))  # 0.05235988
print(math.Radians(-3)) # -0.05235988
print(math.Radians(0))  # 0.0
```
