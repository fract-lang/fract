# ``Capitalize`` function

## Description
Converts the first character to uppercase.

## Define
```
protected func Capitalize(str)
```

## Parameters
+ ``str`` <br>
String.

## Examples
```
open std.strings

print(strings.Capitalize("Fract language"))  # Fract language
print(strings.Capitalize("fract language"))  # Fract language
print(strings.Capitalize("*fract language")) # *fract language
```
