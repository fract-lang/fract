# ``IsLower`` function

## Description
Returns true if string is lowercase, false if not.

## Define
```
protected func IsLower(str)
```

## Parameters
+ ``str`` <br>
String.

## Examples
```
open std.strings

print(strings.IsLower("a")) # true
print(strings.IsLower("A")) # false
print(strings.IsLower("*")) # false
```
