# ``IsUpper`` function

## Description
Returns true if string is uppercase, false if not.

## Define
```
protected func IsUpper(str)
```

## Parameters
+ ``str`` <br>
String.

## Examples
```
open std.strings

print(strings.IsUpper("a")) # false
print(strings.IsUpper("A")) # true
print(strings.IsUpper("*")) # false
```
