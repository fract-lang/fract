# ``Lower`` function

## Description
Returns strings as lowercase.

## Define
```
protected func Lower(str)
```

## Parameters
+ ``str`` <br>
String.

## Examples
```
open std.strings

print(strings.Lower("Fract"))               # fract
print(strings.Lower("Fract-languaGE"))      # fract-language
print(strings.Lower("3* ER"))               # 3* er
print(strings.Lower("FOOBAR"))              # foobar
```
