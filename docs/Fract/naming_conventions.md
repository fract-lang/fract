# Naming Conventions

A naming cannot begin with a number and a period. It cannot end with a point. It cannot contain any special characters except periods and underscores.

## Wrong Name Examples
+ .name
+ name.
+ 45name
+ 0name

## Correct Name Examples
+ name
+ name_
+ _name
+ na.me
+ na_me
+ name.1
+ n4m3
+ na._me
